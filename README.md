Human in the Loop

AI is a personal journey. Let's begin.

A personal AI mentorship landing page — built to connect people who feel left behind by AI with a real human who can help them find their starting point.
Live site → human-in-theloop.netlify.app

What it is
Human in the Loop is a one-on-one AI mentorship service. No courses, no chatbots, no overwhelming frameworks — just a single human conversation to help you figure out where AI fits in your life and work.
This repo contains the landing page for that service.

Built with

Vanilla HTML, CSS & JavaScript — no frameworks, no dependencies
Canvas API particle constellation with mouse gravity physics
CSS-animated loop visual with orbiting dot and SVG human figure
Film grain + radial vignette overlays via CSS pseudo-elements
Ghost word parallax on mousemove
Google Apps Script web app as form endpoint
Google Sheets as the data store
Google Analytics (GA4)
Open Graph + Twitter Card meta tags
Deployed on Netlify


Why I built this
I use AI every day — at work as a Product Manager and in my personal life. I kept noticing that the people around me weren't using it, not because they didn't want to, but because they didn't know where to start and felt embarrassed to ask.
Human in the Loop is my answer to that. A real human, a real conversation, a real starting point.
